# CLI Password Manager

## Features
- Add, view, update, delete passwords
- Search passwords by name/service
- AES-256 encryption for storage
- Master password protection
- Export to encrypted file
- Import from backup

## Tech
- Language: Python 3.10
- Encryption: cryptography library
- Storage: SQLite with encryption
- CLI: Click framework
- Config: JSON files
